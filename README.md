# JSONPlaceholder API Testing

## Overview

This project demonstrates API testing using Postman against the JSONPlaceholder public REST API.

## Tools Used

- Postman
- REST API
- JSONPlaceholder

## Test Scenarios Covered

### GET Requests
- Retrieve Users
- Retrieve Posts

### POST Requests
- Create New Post

### PUT Requests
- Update Existing Post

### DELETE Requests
- Delete Post

## Validations Performed

- Status Code Validation
- Response Time Validation
- Response Body Validation
- JSON Field Validation

## Repository Contents

- Postman Collection
- API Test Scripts
- Request Validations
